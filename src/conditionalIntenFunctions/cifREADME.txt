README for folder conditionalIntenFunctions

DESCRIPTION

    Here in this folder are the necessary classes and functions necessary to create, train, and predict neuron spikes
    using conditional intensity functions.

FILES AND FOLDERS

    First we will talk about the three primary classes: (1) condItenFuncManger in cifManager.py (2)
    condInenFuncFilterManger in cifFilterManager.py and (3) condItenFunc in cifNeuronCif.py.
